# 🛠 Backend - API Server

## 프로젝트 소개

Jack's Community 프로젝트 백엔드 리포지터리입니다.

## 📌 기술 스택

-   Node.js (Express)
-   MySQL
-   쿠키와 세션 (로그인 인증)

## 🚀 설치 및 실행

```bash
    git clone https://github.com/100-hours-a-week/2-jack-lee-community-be.git
    npm install
    npm start
```

## 🔑 환경 변수 설정 (.env)

```bash
    DB_HOST = your-database-url
    DB_PORT = your-port
    DB_USER = username
    DB_PASSWORD = password
    DB_NAME = your-database-name
```
